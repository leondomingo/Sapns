Welcome to Sapns!

Sapns is an open source project released under the GPL (don't know yet...) license, and built on top of Turbogears. It pretends to be a one-size, fits-all system for intensive data-management web apps. Regardless the application you want to build, Sapns will help you in that task with a set of tools, so you can manage data (and more) from the very first moment.

(intensive data-management explanation)

Sapns makes use of (and loves) other open projects:

  • Python to make almost everything
  • PostgreSQL to store data
  • SQLAlchemy to define and manage data models
  • Jinja2 to write templates
  • jQuery to improve user interface

Not part of Sapns but very useful too:

  • Apache to serve apps
  • mod-wsgi to connect Turbogears and Apache
  • Eclipse to write apps
  • Git to control source
  • GitHub to make all this code available
